# swasth
Swasth: Post-visit medical assistant. Reads prescriptions (OCR), provides verified e-commerce links for meds, and offers personalized diet/lifestyle advice. Transforming doctor's orders into an actionable, seamless wellness plan.

# Installation
run this command to setup python virtual env and install modules
```bash
python -m venv .venv
source .venv/bin/activate
pip install fastapi uvicorn "jinja2" python-multipart
```
# Running the Server
```bash
python main.py
```
